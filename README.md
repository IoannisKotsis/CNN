This multi-task CNN is designed to classify:
food-related images from social media ads.

- Head-1: multi-class classification (accuracy: 96.13% )
- Head-2: multi-label classification (micro F1 score: 0.92)
- Head-3: binary classification (positive class F1 score: 0.94)

Dataset: 5851 images -> training 70%, validation 15%, testing 15%
